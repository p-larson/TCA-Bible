# TCA-Bible
[![Swift](https://github.com/p-larson/TCA-Bible/actions/workflows/swift.yml/badge.svg?branch=main)](https://github.com/p-larson/TCA-Bible/actions/workflows/swift.yml)

This project is solely a demonstration of a multiplatform `iOS`/`macOS` application built with `SwiftUI` and `TCA`. Although both apps are distinct, they do share a lot of code-base. Using `TCA` to hyper-modularize this project has allowed the two targets to share the most feasible amount of depenedencies, which has been a huge advantage.
